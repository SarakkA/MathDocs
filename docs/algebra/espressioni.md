# Espressioni polinomiali

!!! tip "Suggerimento"
    Prima le potenze, poi le moltiplicazioni e infine somme e differenze

### Esercizi con soluzione

| Espressione | Soluzione |
|-------------------|-----------------|
| $-4x^2 + (2x + 3)^2$ | $12x + 9$ |
| $3x^2 + 2(3x - 1)^2$ | $21x^2 - 12x + 2$ |
| $(2x + 1)(3x + 1) - 6$ | $6x^2 + 5x - 5$ |
| $x + (1 - 2x)(-3x - 4) + (2x - 1)^2$ | $10x^2 + 2x - 3$ |
| $2(-3x - 1)^2 + (-2x - 2)(3x - 3)$ | $12x^2 + 12x + 8$ |
| $4x(2x - 4) - 1$ | $8x^2 - 16x - 1$ |
| $x(3x - 2) + (-x - 3)^2 - 3$ | $4x^2 + 4x + 6$ |
| $-6x^2 + (3 - 3x)(x - 2)$ | $-9x^2 + 9x - 6$ |
| $(4 - 2x)(4 - x) + (-x - 1)^2$ | $3x^2 - 10x + 17$ |
| $(3 - x)^2 + (-x - 1)(3x + 1)$ | $-2x^2 - 10x + 8$ |
