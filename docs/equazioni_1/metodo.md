# Metodo risolutivo

Per risolvere un'equazione di primo grado:

1. Portare tutti i termini contenenti l'incognita su un lato.
2. Portare i termini numerici sull'altro lato.
3. Semplificare i termini simili.
4. Dividere per il coefficiente dell'incognita.

=== "Equazione 1"

    $$4x + 3 = 3x + 9$$ 

    Sottraggo $3x$

    $$4x + 3 -3x= 3x + 9-3x$$ 

    $$x + 3 = 9$$ 

    Sottraggo $3$

    $$x + 3 -3 = 9 -3$$ 

    $$x= 6$$ 



=== "Equazione 2"

    $$ 3(x+4)=6+x $$ 

    Semplifico le parentesi

    $$3x+12=6+x$$

    Sottraggo $x$

    $$3x+12-x=6+x-x$$

    $$2x+12=6$$

    Sottraggo $12$

    $$2x+12-12=6-12$$

    $$2x = -6$$

    Divido per 2

    $$\dfrac{2x}{2} = \dfrac{-6}{2}$$

    $$x = -3$$
