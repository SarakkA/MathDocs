# Equazioni di secondo grado spurie

Le equazioni spurie hanno la forma:

$$ax^2 + bx = 0$$

Si procede prima **raccogliendo la \(x\)** poi, grazie alla legge dell'annullamento del prodotto(1), dividiamo in due equazioni
{ .annotate }

1.  !!! theorem "Legge dell’annullamento del prodotto"
        Se \(ab = 0\), allora  
        \(a = 0\) **oppure** \(b = 0\)



=== "Esempio 1"
    
    $$4x^2 + 8x = 0$$
    
    $$x(4x + 8) = 0$$
    
    $$
    \begin{aligned}
    x = 0 \quad \text{oppure} \quad &4x + 8 = 0 \\
    &x = -2
    \end{aligned}
    $$


=== "Esempio 2"
    
    $$-5x^2 + 4x = 0$$
    
    $$x(-5x + 4) = 0$$
    
    $$x = 0 \quad \text{oppure} \quad x = \tfrac{4}{5}$$
