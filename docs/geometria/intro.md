# Esercizi

Risolvi le seguenti equazioni di secondo grado.

## Livello base

1. $x^2 - 3x + 2 = 0$
2. $x^2 + 4x + 4 = 0$

---

## Livello intermedio

3. $2x^2 - x - 3 = 0$
4. $3x^2 + 6x + 1 = 0$

---

## Suggerimento

Per ogni esercizio:
1. individua $a$, $b$, $c$;
2. calcola il discriminante $\Delta$;
3. applica la formula risolutiva.
