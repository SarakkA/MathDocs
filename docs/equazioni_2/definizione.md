# Equazioni di secondo grado

!!! definition "Definizione"
    Un'equazione di **secondo grado** Ã¨ un'equazione riconducibile alla forma  
    
    $$ax^2 + bx + c = 0 \qquad a \neq 0$$

=== "Esempio 1"
    $$3x^2 + 5x + 1 = 0 \quad \Rightarrow \quad a = 3,\ b = 5,\ c = 1$$

=== "Esempio 2"
    $$-4x + 2 + x^2 = 0 \quad \Rightarrow \quad a = 1,\ b = -4,\ c = 2$$

=== "Esempio 3"
    $$6x^2 + 2x = 0 \quad \Rightarrow \quad a = 6,\ b = 2,\ c = 0$$

=== "Esempio 4"
    $$\frac{x^2}{3} - 1 = 0 \quad \Rightarrow \quad a = \frac{1}{3},\ b = 0,\ c = -1$$

=== "Esempio 5"
    $$2x^2 = 0 \quad \Rightarrow \quad a = 2,\ b = 0,\ c = 0$$

=== "Esempio 6"
    $$8x^2 = 3x - 1 \quad \Rightarrow \quad a = 8,\ b = -3,\ c = 1$$

!!! theorem "Classificazione"
    Un'equazione di secondo grado si dice:
    
    - **pura** se \( b = 0 \)  
    - **spuria** se \( c = 0 \)  
    - **completa** se \( b \neq 0 \) e \( c \neq 0 \)
